# Inspr (Atom)

Inspr 是一个程序员命名解决方案，以插件的形式集成在主流编辑器中，专治各种命名头疼患者。

## 食用方式

输入中文，按下 Inspr 翻译快捷键，列出可选的命名方案。

例如 `let 军团（Ctrl + Command + i）`，弹出上下文联想菜单

```
corp, Corp, faction, Faction ....
```
